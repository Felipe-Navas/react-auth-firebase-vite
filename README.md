## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/react-auth-firebase-vite.git && cd react-auth-firebase-vite && npm install
```

### Executing program

```
npm start
```
